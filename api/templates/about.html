About

Hi, I'm Chris. I built UK Bin Day to make finding out your bin days a little easier. It provides a search box for ~85% of UK councils. You can subscribe to a live calendar feed that keeps itself up to date as your council publishes new collections, so you never forget again! Please reach out to hello@ukbinday.com if you have any thoughts or suggestions!

This project builds on the work of two open-source projects designed for Home Assistant automation. The contributors to these two projects have done the seriously hard work of figuring out each individual councils website:

hacs_waste_collection_schedule UKBinCollectionData

Thank you to the maintainers and contributors of both projects.

API

A free API is available for the Home Automation community, and whoever else finds this useful. The UPRN endpoint is not publicly exposed, to prevent abuse. See the API documentation for details.
