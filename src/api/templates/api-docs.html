API Documentation

This project exposes a public REST API for looking up UK bin collection schedules.

Interactive Docs

Swagger UI (OpenAPI) ReDoc

Endpoints

  • GET /api/v1/addresses/{postcode} - Look up addresses for a postcode
  • GET /api/v1/lookup/{uprn} - Get bin collection dates
  • GET /api/v1/calendar/{uprn} - Subscribe via .ics calendar
  • GET /api/v1/councils - List supported councils
  • GET /api/v1/health - Scraper health status

OpenAPI Schema

The raw OpenAPI JSON schema is available at /api/v1/openapi.json.
